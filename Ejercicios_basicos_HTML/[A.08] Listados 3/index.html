Listado de palabras:

Variable
Un nombre simbólico asociado a una ubicación en la memoria que contiene un valor. Este valor puede cambiar durante la ejecución del programa.
Función
Un bloque de código reutilizable que realiza una tarea específica. Puede recibir datos de entrada (parámetros) y devolver un resultado.
Compilador
Un programa que traduce el código fuente escrito en un lenguaje de programación de alto nivel a código máquina (instrucciones que la computadora puede ejecutar directamente) antes de la ejecución.
BIOS
Firmware que inicializa el hardware antes de arrancar el sistema operativo.
Git
Sistema de control de versiones que permite registrar los cambios realizados en el código fuente.
HTML
Lenguaje de marcado para estructurar el contenido de páginas web.
