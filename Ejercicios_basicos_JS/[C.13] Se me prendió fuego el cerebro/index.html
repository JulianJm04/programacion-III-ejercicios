[C.13] Se me prendió fuego el cerebro Definir la función obtenerSumaMaxima(arr). Recibirá cómo parámetro un array de números, por ejemplo: [1, -2, 3, 4, -9, 6]. La tarea es encontrar, dentro de ’arr’, el subarray de elementos contiguos que tenga la suma máxima. Por ejemplo: obtenerSumaMaxima([-1, 2, 3, -9]) //5 obtenerSumaMaxima([2, -1, 2, 3, -9]) //6 obtenerSumaMaxima([-1, 2, 3, -9, 11]) //11 obtenerSumaMaxima([-2, -1, 1, 2]) //3 obtenerSumaMaxima([100, -9, 2, -3, 5]) //100 obtenerSumaMaxima([1, 2, 3]) //6 Nota: si todos los valores son negativos, no se toma ningún elemento y su suma es cero. obtenerSumaMaxima([-1, -2, -3]) // 0
