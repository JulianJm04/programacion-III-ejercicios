Enunciado:

[D.01] Producto JSON Crear, dentro del archivo producto.js, un JSON para representar información acerca de un producto (código de barra, nombre, precio y fecha de vencimiento). Asignarle valores a cada una de sus propiedades. Validar su buen diseño ingresando en http://jsonviewer.stack.hu/. Una vez validado el JSON, mostrar todas sus propiedades por consola.
